# 快速开始 - 写作Agent指南

欢迎使用写作Agent！这是一个基于两层判断机制、9步工作流的结构化AI写作助手。

---

## 🚀 30秒快速上手

### 三步开始写文章：

1. **准备Brief** （或口头说明需求）
2. **告诉我想写什么** → "帮我写一篇公众号文章，关于..."
3. **我会自动执行完整流程** → 从搜索、选题、写作到审校

---

## 📚 完整工作流一览

### 如果你什么都不想管，就这样用：

```
你: "帮我写一篇技术文章，介绍[话题]"
   ↓
Agent执行完整流程:
  1️⃣ 搜索最新信息
  2️⃣ 提供3-4个选题方向（你选一个）
  3️⃣ 如果需要，创建测试清单
  4️⃣ 学习你的写作风格
  5️⃣ 整合你的个人素材（有的话）
  6️⃣ 等待你的测试数据（如需要）
  7️⃣ 写出初稿
  8️⃣ 三遍审校（删AI腔）
  9️⃣ 配图完成
   ↓
你: 收到完整、可发布的文章
```

## 🎯 准备工作（可选但推荐）

### 最重要：建立个人素材库

**为什么**：降AI味的关键。文章中加入你的真实经历、观点、案例。

**怎么做**：
1. **导出即刻动态**（如果你用过即刻）
   - 保存为CSV
   - 放到 `_素材库/原始数据/`

2. **整理项目案例**
   - 你做过什么项目？
   - 取得了什么成果？
   - 花了多少时间？

3. **收集个人笔记**
   - 你的思考和洞察
   - 真实的观点
   - 学到的经验

**结果**：Agent写文章时会自动引用你的真实素材，文章读起来像你本人。

### 其次：填写风格指南（可选）

在 `CLAUDE.md` 中补充你的个人风格：

```markdown
### 我的写作风格

语言特征：
- 常用词："真的"、"其实"、"确实"
- 避免："显著"、"充分"、"有效地"

开头方式：
- 喜欢用故事引入，而不是问题
- 直奔主题，少铺垫

句子长度：
- 15-25字为主，避免超过30字

其他特点：
- 有观点，不中立
- 有温度，不冷硬
```

## 🆘 常见问题

### Q: Agent会直接写文章吗？

**A**: 不会。它会先搜索信息，再提供3-4个选题方向让你选。你选了才继续。这样能避免方向错误。

### Q: 为什么一定要三遍审校？

**A**:
1. 第一遍检查逻辑和事实
2. 第二遍删AI腔（套话、句式、表达）
3. 第三遍打磨细节（标点、排版、节奏）

系统化降低AI检测率，确保文章质量。

### Q: 个人素材库怎么建立？

**A**:
1. 导出即刻动态或笔记
2. 保存到 `_素材库/原始数据/`
3. 建立分类索引
4. Agent写文章时自动引用

不需要很多，真正值得用的就那点。

### Q: 如果很急，能跳过步骤吗？

**A**: 可以，但要明白风险：
- 跳过选题讨论 → 可能方向错误
- 跳过调研 → 可能信息过时或错误
- 跳过三遍审校 → 质量降低，AI味重

核心原则（Think Aloud、不编造、三遍审校）强烈建议不要跳。

### Q: 为什么要搜索？AI训练数据不够吗？

**A**: 不够。AI训练数据通常停留在2024年初左右。如果写2024-2025年的新工具、新技术，必须搜索最新信息。搜索验证 > 一切。

### Q: 什么是"Think Aloud"？

**A**: Agent说出它的思考过程。

❌ 不好："这个选题最好"
✅ 好："对比了这三个选题，我觉得这个最适合，因为1️⃣...2️⃣...3️⃣..."

这样你能看到AI在想什么，及时纠正。
